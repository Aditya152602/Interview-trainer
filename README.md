## ⚡ What Makes This Different

Most interview prep tools give you the same 50 questions everyone else studies.

**Interview Trainer Agent doesn't.**

Every session is freshly generated by a 70-billion parameter AI model that understands your exact role, experience level, and industry — then scores your actual answers, not just checks if you memorized the right words.

```
You type:  "Java Developer | 3 years | FinTech"
AI builds: 30 tailored questions across 4 categories
You answer → AI scores → You improve → You get hired
```

## 🧠 Core Features

### 🎯 Intelligent Question Generation
- **30 questions per session** tailored to your exact role and experience
- **4 question types** — Technical, Behavioral (STAR), Situational, HR & Culture
- **3 difficulty tiers** — Easy, Medium, Hard — distributed intelligently
- Questions change every session — no repetition, no memorization

### 🤖 Real-Time AI Evaluation
- **Score 1–10** with detailed reasoning for every answer
- **Strengths identified** — know what you're already doing right
- **Improvement areas** — specific, actionable, not vague
- **Model answer** — see exactly what a top-scoring response looks like
- **Pro tip** — one coaching insight per question to sharpen your style

### ⏱️ Built for Realistic Practice
- **Live timer** per question — mirrors real interview pressure
- **Word counter** — trains you to give complete, thorough answers
- **STAR format prompt** — automatic reminder for behavioral questions
- **Session history** — full breakdown at the end with per-question scores

### 🔐 Secure by Design
- API key lives only in Vercel's environment — never in the browser
- No user data stored, no accounts required
- Zero tracking, zero ads

## 🏗️ How It Works

```
┌──────────────────────────────────────────────────────────────┐
│                        USER BROWSER                          │
│                    React + Vite Frontend                     │
│          (No API keys, no secrets, just UI logic)           │
└─────────────────────┬────────────────────────────────────────┘
                      │  POST /api/chat
                      │  { system prompt + messages }
                      ▼
┌──────────────────────────────────────────────────────────────┐
│                    VERCEL EDGE FUNCTION                      │
│                       api/chat.js                            │
│         (API key secured as environment variable)           │
└─────────────────────┬────────────────────────────────────────┘
                      │  Authenticated request
                      ▼
┌──────────────────────────────────────────────────────────────┐
│                       GROQ CLOUD API                         │

│              (~500 tokens/sec · <1s latency)                 │

```

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- A free [Groq API key](https://console.groq.com)
- A free [Vercel account](https://vercel.com)

### Local Development

```bash
# 1. Clone the repo
git clone https://github.com/Aditya152602/Interview-trainer.git
cd Interview-trainer

# 2. Install dependencies
npm install

# 3. Set up environment
cp .env.example .env
# Add your Groq key → GROQ_API_KEY=gsk_...

# 4. Install Vercel CLI
npm install -g vercel

# 5. Run locally (frontend + API together)
vercel dev
# → Open http://localhost:3000
```

### Deploy to Production

```bash
# Push to GitHub
git add .
git commit -m "Deploy interview trainer"
git push origin main

# On Vercel:
# 1. Import GitHub repo
# 2. Add env var: GROQ_API_KEY = gsk_...
# 3. Deploy → live in 30 seconds
```

## 🔑 Environment Variables

| Variable | Description | Where to get |
|----------|-------------|--------------|
| `GROQ_API_KEY` | Groq API authentication key | [console.groq.com](https://console.groq.com) → API Keys |

> **Security note:** This key never reaches the browser. It lives exclusively inside Vercel's serverless runtime.
